Pattern Cards

Print and cut into cards with one pattern piece per card. Try to arrange these cards to generate the Frieze Patterns.


Printing instructions:

Print as double sided. The print dialog should arrange the patterns to have one set of patterns per page. The pages should line up so that the two sides of patterns are perfect mirrors. Works best using Firefox. Mess around with the width of the page - make it wider and narrower - so that the pattern rows format to your paper size.
