We need a better way to talk about this. We say a shape has rotational symmetry of order n if it has n unique rotations. This means the shape can be rotated of the way around the circle without changing its appearance.

...

For example, a triangle can be rotated by of the way around the circle so a triangle has order 3. A square can be rotated of the way around the circle without changing, so it has order 4.

We can say the same for shapes that are not regular polygons.
